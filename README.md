# elastic-textarea 

```javascript
var elasticTextarea = new ElasticTextarea(element);
```

## Options

```javascript
var textarea = $("textarea")[0],
    options  = { id: "textarea-1", class: "textarea-wrapper" };

textarea.elasticTextarea = new ElasticTextarea(textarea, options);
```
